## ğŸ¯ Project Overview

**Pushpaka Vimana** is an open-source micro-UAV research project inspired by ancient Indian aeronautical imagination and executed strictly using **modern aerospace engineering principles**.

Named after the legendary *Pushpaka Vimana* from Sanskrit literature, this project aims to **bridge cultural inspiration with practical engineering**, resulting in a reproducible **1:10 scale flying prototype**.

The project emphasizes **feasibility, safety, transparency, and education**, not speculative or unsafe propulsion claims.
