## ğŸ¯ Project Overview

**Pushpaka Vimana** is an open-source micro-UAV research project inspired by ancient Indian aeronautical imagination and executed using modern aerospace engineering principles.

Named after the legendary *Pushpaka Vimana* from Sanskrit literature, this project aims to **bridge cultural inspiration with practical engineering**, resulting in a reproducible 1:10 scale flying prototype built within 60 days.

The project emphasizes **feasibility, safety, transparency, and education** rather than speculative or unsafe propulsion claims.

## âš¡ Power & Propulsion Strategy

### Primary Propulsion

* Brushless DC motor + propeller
* ESC-controlled thrust
* LiPo-powered (safe discharge limits)

### Solar Integration (Auxiliary)

* Flexible thin-film solar layer
* Used for:

  * Telemetry support
  * Sensor power
  * Ground charging experiments

### Argon Research Module (Non-Propulsive)

* **Strictly experimental**
* Low-energy plasma/ionization sensor study
* No thrust generation
* Educational & diagnostic purpose only
